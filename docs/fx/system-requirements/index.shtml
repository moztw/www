Firefox 系統需求

Windows

作業系統版本
  • Windows 2000
  • Windows XP
  • Windows Server 2003
  • Windows Vista
硬體最低需求
  • Pentium 233 MHz CPU（建議 Pentium 500MHz 以上）
  • 64 MB RAM（建議 128 MB 以上）
  • 52 MB 硬碟空間

Linux

軟體需求

請留意：各 Linux 發行版隨附的 Firefox 套件可能有其不同需求。

  • Firefox 需要以下函式庫以正常執行：
    • GTK+ - 2.10 或以上版本
    • GLib - 2.1.2 或以上版本
    • Pango - 1.14 或以上版本
    • X.Org - 1.0 或以上版本

  • 我們建議安裝以下函式庫及套件，以取得最佳效能：
    • NetworkManager - 0.7 或以上版本
    • DBus - 1.0 或以上版本
    • HAL - 0.5.8 或以上版本
    • GNOME - 2.16 或以上版本

Mac

作業系統版本
  • Mac OS X 10.4 以上
硬體最低需求
  • 使用 Intel x86 或 PowerPC G3、G4、G5 CPU 之麥金塔電腦
  • 128 MB RAM（建議 256 MB 以上）
  • 200 MB 硬碟空間
