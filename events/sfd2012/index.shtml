2012 MozTW SFD Party

Code Rush 源碼奔馳 放映會

簡介

九月的第三個星期六，是軟體自由日（Software Freedom Day，SFD） 在關於自由軟體和開源軟體的全球性慶祝活動當中，我們預計在電影院上演當初手提一只皮箱就跑進的矽谷淘金的工程師們愛與勇氣的冒險記-Code Rush（源碼奔馳）。

活動資訊

  • 時間：5/27 下午 4:40 ~ 6:50
  • 地點：西門町-真善美劇院（台北市萬華區漢中街116號7樓）
  • 費用：200 元（先報名、現場付費取票）

交通

  • 捷運站六號出口出來正前方
  • 誠品左手邊的樓上
