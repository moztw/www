什麼是 Mozilla Firefox Party？
Firefox 1.0 正式版即將出爐，這對於 Firefox 而言是一個很重要的里程埤，Mozilla 官方為了慶祝這歷史性的一刻， 決定邀請全世界一同來辦一場屬於 Firefox 的 Party，於是展開了全球的串聯，這樣盛大的聚會，我們台灣怎麼可以缺席呢? 因此，台灣 Mozilla 社群決定在台灣辦一場盛大的聚會，就讓我們一同來傳遞 Firefox 的聖火到全台灣吧!!
除了 party 之外，還有活動!!
您以為在台灣只有辦 party 嗎? 為了台灣廣大的 Firefox 支持者，我們還特地舉辦了三場從基礎到進階的研討會(免費)， 想知道你手邊的 Firefox 到底有什麼魔力? 那麼，這三場研討會絕對適合您，不論是新手、老手、高手，全部都適合， 這樣的機會怎麼能放過呢? 詳情
有更多的活動?!
沒錯! 有更多的活動，不只有研討會和 Party，在 Party 當天之外，我們還外加推廣週、放火行動、祝福話全台簽名等活動，請拭目以待吧!!
除了台北，其他地區也有辦 party 嗎?
只要 Mozilla Party2 上有您所屬的地區，您也可以報名參加。如果，您想自己邀三五個好友， 也是可以的，請參考[推廣活動]的頁面。
11月21日當天活動，須要付什麼費用嗎?
11月21日當天，研討會是不需要付費的，只要人來就好，而我們的 Party 可以算是"聚餐" + "聚會"個人的吃喝由個人負擔， 再加上我們包場的費用，我們先統一收 300 元，如果有多我們會退，如果您個人想吃更好一點，也可以多付一些費用，整天 活動下來 300 元，可以說是很划算哦!!因為研討會的費用我們幫您吸收了 : )
