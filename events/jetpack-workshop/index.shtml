加入 Jetpack 三倍速套件開發行列！

Jetpack 是 Mozilla Labs 正在積極研發，將用於 Firefox 的新擴充套件開發框架，提供超簡單而更快速的套件開發、維護和散佈方案。只需要會簡單的 HTML 和 JavaScript，就可以輕鬆加入開發的行列。想了解目前的 Jetpack 框架和即將推出的 Jetpack Reboot 新結構嗎？

就讓「Jetpack 大使」小B 帶領您，一步一步學習三倍速的套件開發，並動手實做自己的 Jetpack 擴充套件！

本課程適合對 Firefox 套件開發有興趣、具備基本 JavaScript 程式設計能力者。也歡迎想踏入瀏覽器擴充套件領域的網頁設計師參加。

講師介紹

尤孝庭 ( 笨笨的小B / ybite )，台大電機系就學中，長期研究網頁相關技術，是 Mozilla 首期「Jetpack 大使」的成員。

擴充套件作品有 Voyage 及 nicoFox 等，其中 Voyage 獲得 Extend Firefox 3.5 Contest "Grand Prize" 優選。
