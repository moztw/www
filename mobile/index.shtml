輕鬆 個人化

保持同步

無縫同步您電腦和手機上的 Firefox，如此一來，您就可以隨時隨地存取您的瀏覽紀錄、密碼、書籤，甚至是您開啟的分頁。在您第一天打開手機的時候就可以存取多年來在電腦上瀏覽的紀錄，並使用從電腦上儲存下來的密碼在您的手機上填寫表單。

您現在可以在電腦瀏覽器中瀏覽網頁，在瀏覽的過程中離開一下就會發現這些分頁已經在手機上等著您，就跟您離開電腦的時候一模一樣。 您的瀏覽體驗將會煥然一新！了解更多資訊。

行動附加元件

想讓 Firefox 投您所好、成為您的專屬瀏覽器，就瞧瞧我們不斷茁壯的附加元件目錄，在手機上直接瀏覽、安裝附加元件！任何開發者都可以為行動版 Firefox 打造專屬附加元件，或者直接改造目前現有的套件、以符合行動瀏覽的需求。

Awesome Screen：看得多、打得少

在行動裝置上輸入文字不是件方便的事，這也是我們創造 Awesome Screen 的理由。隨著您瀏覽的次數越來越多，它也會越來越懂得您的需求，視情況提供您最常去的網站、過往加過書籤的網頁、或者是在其他電腦上開過的分頁。只要點選位址列就行了！

搜尋引擎

為了要有最好的搜尋效果，當您開始在 Awesome Bar 輸入時就會出現多個搜尋引擎。您也可以點一下 Awesome Bar 旁邊的放大鏡來看看您的搜尋引擎選項，或是透過附加元件管理員來自訂該清單。

專屬啟動頁

Firefox 的個人啟動頁讓您即刻取用在桌機開啟的分頁、上次在手機上瀏覽的網頁、以及其他推薦使用的附加元件。

妝點您的 Firefox

您也可以用行動版 Firefox 取得上千種可以輕鬆安裝的佈景主題（Personas），輕輕一點就能讓瀏覽器改頭換面。今天要換上規律的花色、自然美景，還是電影主題如哈利波特、時尚精選如 Anna Sui？任您挑選！

友善的 觸控螢幕支援

只要瀏覽，不要凌亂

您的螢幕上可能已經沒有空間可以塞下更多東西了，所以我們把 Firefox 設計成不影響您目前所在閱覽的內容。當您打開某個網址之後，Awesome Bar 與畫面上的控制項就會被藏起來，但您可以很快地再找到它們：只要把畫面往左或往右滑動，就可以看到您的分頁與瀏覽器工具；或是往上滑動以開始新的搜尋。

縮放！

在小小的畫面當中閱讀細小的文字會讓眼睛脫窗。若您想要快速的放大或縮小文字，只需要在您想要看到更大的文字的時候隨時雙指縮放（pinch-to-zoom）（於 Android）或雙擊螢幕（於 Android/Maemo）。

分頁瀏覽

已經在您的電腦上享受過分頁瀏覽的方便了嗎？Firefox 行動版也有分頁功能！您只需要向右滑動畫面就可以看到所有已開啟分頁的圖示，您就可以簡單地看出接下來所要切換到的分頁。點一下加號的按鈕以開啟新分頁，若您不小心關掉了某個分頁，也可以點一下復原按鈕把它找回來。

一觸加入書籤

把畫面往左滑動並把您喜愛的網站加入書籤。接下來您就可以幫它加上標籤，如此一來您就可以在 Awesome Screen 上快速地存取該網站。

生產力 與效率

速度超快

Firefox 行動版是建立於跟電腦版的相同引擎，同時也已經對在行動裝置上的瀏覽作過最佳化。結果就是速度超快、超有效率，以及無縫的瀏覽體驗。

位置感知

想要找找這附近有什麼好吃的，或是通往高速公路的捷徑嗎？Firefox 可以透過向您的手機詢問您身在何處以節省您的時間。這很好用，但這個功能是全然自願性的 – Firefox 只會在您同意的時候才會分享您的所在位置。了解更多資訊。

下載管理員

就跟您使用電腦桌面上的 Firefox 一模一樣，就算是您一次下載多個檔案，檔案都會在背景中被下載，所以不會打斷您的瀏覽。另外，下載管理員可以讓您隨時停止、暫停，或恢復下載過程。

網站選單

使用行動裝置不代表您也會失去您瀏覽過程的控制性。點下網站 ID 的按鈕（在網址或網站標題旁邊的圖示）就可以取得關於您所瀏覽的網頁的一系列選擇。視網站而定，您可以在網站選單當中看到下列選項：

  • 儲存成 PDF

    儲存成 PDF 功能讓您可以留下地圖、登機證等重要的內容以離線檢視。

  • 忘記密碼

    忘記密碼功能可以讓網站忘記您的私人資料。

  • 新增搜尋引擎

    新增搜尋引擎可以讓您新增一個搜尋引擎到您的自訂搜尋引擎擎端。

  • 頁面搜尋

    頁面搜尋功能讓您可以快速掃過頁面上的文字，找到您想要的資料。

  • 分享頁面

    頁面分享功能讓您可以把資料送到電子郵件信箱、Facebook、Twitter、Google Reader，以及其他更多地方。

  • 清除網站偏好設定

    清除網站偏好設定可以處理特定網站的偏好設定，包含彈出型視窗、自動分享地理位置資訊、本機資料儲存，以及永遠不要儲存密碼的選擇。

自動更新

Firefox 將會在有更新、更好的版本可用時通知您。您可以馬上更新，或是準備好後再更新。

密碼管理員

Firefox 可以記得您最愛網站當中的密碼，再也不必重複又難又煩的輸入動作。

進階 安全性

安全性偏好設定

我們非常看重您在網路上的安全，也把 Firefox 包得緊緊地以保護您的安全。向左滑動畫面並點下瀏覽器工具以開啟偏好設定畫面。這個畫面可以簡單地讓您自訂您密碼、個人資料、Cookies、圖片載入，以及安裝附加元件的設定。

即時檢視網站身分資訊

想要在購物前再次確認網站身分嗎？按下網站圖示立刻顯示網站身份資訊。如此可以避開詐騙及有害網站，讓您確定網站的真實身分。

忘記密碼

您有時可能會想要 Firefox 停止記憶您的密碼。若您正在分享您的手機，或單純不想再相信某個網站，只要使用網站選單當中的忘記密碼設定就可以清除這些資訊。

尖端技術

HTML5

Firefox 使用像是 HTML5, CSS 與 JavaScript 的開放規格，所以開法者可以建立快速、強大，以及美觀的應用程式。也有相機等行動裝置上的特殊功能，像是地理位置 與 裝置方位偵測。了解更多資訊。

音訊與視訊

HTML5 音訊與視訊支援能為您的手機帶來更豐富的瀏覽體驗。您現在可以觀賞諸如 WebM 或 Ogg Theora 等開放格式的影片。立刻行動，在您的手機看看我們的 Meet Mobile 影片。

電解化

電解化（Electrolysis）可以將瀏覽器介面執行於不同的行程（Process）當中，讓其與描繪中的網頁內容拆分，給使用者一個更穩定及更有響應程度的瀏覽器。

JaegerMonkey JavaScript 引擎

Firefox 包含了一個超快的 JavaScript 引擎，使用了 Mozilla 的 "JaegerMonkey" just-in-time 編譯器，同時也已經為 ARM 處理器作過最佳化。

Layers

Layers 技術可以幫助加強圖形效能，提供更快更順暢的捲動、縮放，及動畫。

離線瀏覽

Firefox 的離線瀏覽功能可以讓您在手機沒有網路連線的時候還是可以隨手取得想要的資訊。某些具備離線功能的網站，像是網頁郵件，可以讓您隨時隨地讀取訊息。
