參與我們

我們想要讓網路變得更好。歡迎參與我們的任務，成為推廣網路自由的一份子！

您不一定得是專業的程式設計師，只要有個熱情的心，願意自己探索該任務的工作內容。歡迎來 MozTW 貢獻自己的時間與心力，開創散播自由的無限可能！目前正在進行下列工作，誠徵同好輔助或是接手以下工作 —— 不用怕自己的能力不符要求，我們完全接受 on-the-job training！

文件專案 / 技術支援服務

整理說明文件，提供軟體使用問題方面的協助；對一般使用者的第一線！

  • 討論區技術支援 - 使用者技術支援長期招募中：回答新使用者在 MozTW 討論區上詢問的軟體使用問題 －－ 是的，不需向我們報名，只要您有回過文章，您就已經參與我們了！
  • 討論區技術支援 - 軟體除錯長期招募中：確認使用者回報的軟體問題，向開發方提出錯誤報告（英文或中文；可以使用中文由在地化經理轉介）
  • 文件專案 - Mozilla Links 中文版「機動隊隊長」徵才中：規劃 Mozilla Links 部落格文章主題，協調翻譯與校訂。
  • 文件專案編輯 - Mozilla Links 中文版長期招募中：翻譯與校訂文章內容；使用 Mozilla Links Editors 郵件群組和其他編輯溝通。
  • 文件專案 - Mozilla Support 與 MozTW Wiki 管理徵才中：整理說明文件與常見的技術支援需求；轉移主網站上的舊內容。
  • 討論區管理員長期招募中：整理討論區內容，移動/移除不適合的文章，執行管理規則。

活動企劃

規劃 MozTW 的實體與線上推廣活動。

  • 活動企劃 - 單一與例行活動徵才中：處理 MozTW 實體活動的地點、活動內容等安排。例如 MozTW at COSCUP 2009、MozTW 連續聚、MozTW Labs
  • 活動企劃 - Release 規劃：規劃新版軟體釋出時需要的相關實體與線上宣傳活動，例如 Firefox 3 狐迷同樂會。
  • 推廣活動企劃：社群推廣活動規劃及執行，專營網路社群低成本、高歡樂的活動。

  • Localization project owner - Firefox and Firefox for Mobile
  • Localization project owner - Thunderbird

      5. Ability to communicate with Mozilla personnel in English at Bugzilla and mailing lists.
      6. Willingness to give presentations to the OSS community at large when required.
  • Localization project owner - CaminoVacant
    • Refer to above for requirements.
      1. Also, knowledge of Mac software localization is required.
  • Localization project owner - addons.mozilla.org
  • Localization project owner - Mozilla websitesVacant
    • Refer to above for requirements.

美術設計

為所有網頁與出版物設計漂亮的畫面；妝點大家的螢幕與生活。

  • 美術設計/互動設計 - 網站徵才中：MozTW 主網站、抓火狐、其他活動頁面
    • 需求:
      1. 願意和網頁設計師合作，建立符合需求的互動設計
      2. 願意在 UI 互動測試後重新改善設計
  • 美術設計 - 平面/圖形宣傳物長期招募中：配合實體活動製作
  • 美術設計 - 個性面版「Personas」長期招募中：製作人人喜愛的個性面版，讓火狐穿新衣！
  • 美術設計 - 狐耳摩莎長期招募中：為 MozTW 吉祥物「狐耳摩莎」設計各種活動用的圖像。需以 CC 授權釋出著作。

Developer / System Admin

開發與技術傳教；網站系統維護。

  • Add-on Developer: 開發 Mozilla 程式的擴充套件；舉辦開發講座時擔任講者。
  • Computer system administrator: 維護 MozTW 網站主機的順利運作。
    • Requirement:
      1. Ability to work on POSIX shell.
      2. Experience in Ubuntu/Debian or FreeBSD server administration.
  • Web developer - gfx.tw: 協助開發社群行銷計畫「抓火狐」的相關功能。
    • Requirement:
      1. Knowledge of web development on the LAMP platform.
      2. Intermediate skills of JavaScript, PHP, and any of the MVC model frameworks (e.g. CodeIgniter, CakePHP)
      3. Willingness to read through English documentations and issue lists when debugging.
      4. Experience and knowledge in building a user-generated content website, with understanding to methods in boosting user participation.
      5. Optionally, the ability to design UI graphics.
  • Web developer - moztw.orgVacant: 協助維護 MozTW 網站
    • Requirement:
      1. Knowledge of various web software, e.g. phpBB, Planet, MediaWiki, etc.
      2. Willingness to be on call for unfortunate emergencies.
      3. Ability to work on POSIX shell.
  • Web developer - other projects長期招募中: 實作其他人企劃的網路服務，或是實做/架設自己的；例如MozTW 自投羅網、協同翻譯站。

如何報名？

大部分的工作都不用報名，做就對了！在地化、網站等相關工作，您需要找到目前負責協調的社群成員，請到 moztw-general 郵件群組自我介紹，並說明您想協助哪些工作。

關於 MozTW

關於 Mozilla Taiwan Community 的緣起、任務與目前主要成員，請參考關於本站頁面。

MozTW 郵件群組

歡迎您至 MozTW 的 Google Groups 郵件討論群組「moztw-general」參與我們的討論。請注意：

  • 這不是技術支援信箱——Firefox 疑難雜症請至討論區發問。
  • 您可以使用任何的信箱訂閱與發言，但網頁介面發言需使用 Google 帳號（這是技術限制）
  • 為了抵禦 spam，首次發言將經過人工審核；請保持耐心，謝謝！
