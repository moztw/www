台灣版 Firefox

「Firefox 台灣版」是美商謀智台灣分公司（Mozilla）所製作的客製化版本 Firefox，以一般版 Firefox 為基礎，透過以下套件加入各種功能。

Firefox 首頁 0.1.7

Firefox 訂製首頁，包含最近瀏覽頁面、最常瀏覽頁面及新分頁等

特別為台灣設計的 Firefox 正體中文版瀏覽器，內建好用附加元件提供更多新功能，以提升使用者體驗。MyFirefox 與國內知名論壇(WIRED, Inside, 癮科技, TechOrange, T客邦, 謀智台客) 合作，每日即時提供您最新科技新知，myFirefox 首頁更兼具入口網站特性，搜尋、天氣、地圖及附加元件介紹小工具等一應俱全，讓您一開瀏覽器便能輕鬆掌握大小事！

Firefox在地化功能提供多樣化分頁選擇。透過新增設的多樣化分頁，使用者可以依照個人使用習慣去設定新分頁開啟的內容，包括熱門網站、推薦網站、熱門造訪頁面。使用者也可以自訂個人化的熱門網站，開啟書籤將己加入書籤的連結拖到要放的九宮格內，或在推薦網站裡，透過貼心的分類，快速找到並瀏覽您感興趣的網站或論壇。

申請 Persona 帳號或 Facebook 登入 MyFirefox ,您就擁有專屬的收藏專頁，可以將你喜愛或暫時無法閱讀的文章加入 myShelf, 方便您日後尋找或閱讀並隨時與朋友分享。

Firefox 捷徑 0.12.6

快速啟動常用軟體

Firefox 捷徑在原來基礎上做了改進，使它更直覺更符合使用者習惯。通過瀏覽器下方的 Firefox 捷徑 小按鈕，可以快速開啓瀏覽網頁時的系統工具，例如我的電腦、小畫家、計算機及記事本…等系統功能。使用者還可以透過網頁截圖功能，輕鬆將完整網頁存成一個圖檔。如果這些還不能滿足你的需求，那麼就自己動手，利用個人化捷徑，設置需要的功能吧。

台灣版工具集 0.1.7

提升使用者體驗

Firefox在地化功能在瀏覽器介面上直接增加了以下 Firefox 常用功能，讓使用者快速便利地使用熟悉的 Firefox 功能。

  • 意見回饋

    輕鬆點擊即可將使用 Firefox 瀏覽器所遇到的問題或建議，回應讓我們知道，請幫助我們做出更符合使用者需求的 Firefox 瀏覽器。

  • 隱私瀏覽模式

    開啟隱私瀏覽功能，能讓您在瀏覽網際網路時，不會被其他使用您電腦的使用者觀看或接觸到您曾造訪過的網站與頁面的所有資料。

  • 網站安全標誌 （綠色安全 / 紅色危險）

    Firefox瀏覽器幫您偵測目前所瀏覽的網站是否安全，隨時保護您安全上網

  • 佈景主題

    你可以在任何時候，按照你的心情、喜好，甚至今天的天氣給你的瀏覽器換衣服！
